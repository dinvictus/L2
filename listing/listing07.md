Что выведет программа? Объяснить вывод программы.

```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func asChan(vs ...int) <-chan int {
	c := make(chan int)

	go func() {
		for _, v := range vs {
			c <- v
			time.Sleep(time.Duration(rand.Intn(1000)) * time.Millisecond)
		}

		close(c)
	}()
	return c
}

func merge(a, b <-chan int) <-chan int {
	c := make(chan int)
	go func() {
		for {
			select {
			case v := <-a:
				c <- v
			case v := <-b:
				c <- v
			}
		}
	}()
	return c
}

func main() {

	a := asChan(1, 3, 5, 7)
	b := asChan(2, 4 ,6, 8)
	c := merge(a, b )
	for v := range c {
		fmt.Println(v)
	}
}
```

Ответ:
```
Вывод: числа от 1 до 8 в различном порядке и количестве, и много нулей, программа не завершится,
так как при выполнении функции asChan каналы закрываются, а при выполнении чтения в бесконечном цикле из закрых каналов,
в канал "c" сначала добавятся некоторые значения, потому что при записи есть случайная задержка, 
а после закрытия какого-либо из каналов, канал "c" начнёт наполняться нулями, так как при чтении из закрытого канала возвращается
нулевое значение типа канала, в данном случае это 0. Помимо нулевого значения, ещё возвращается false, это можно использоваться для
предотвращения таких ситуаций.

```
