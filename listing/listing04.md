Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Вывод будет числа от 0 до 9, но программа не завершится, потому что канал не был закрыт и текущая горутина будет ждать,
пока в канал ch не поступит значение, а анонимная горутина уже завершилась, соответственно данные в канал не поступят,
а main горутина будет заблокирована и произойдёт deadlock

```
